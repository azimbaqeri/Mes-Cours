# 📚 Mes Cours

Bienvenue dans ce dépôt qui regroupe mes différents exercices, projets et travaux pratiques réalisés au cours de ma formation et de mon apprentissage en développement informatique.

## 🎯 Objectif

Ce dépôt a pour but de centraliser les exercices et exemples de code réalisés dans différents langages de programmation et technologies afin de :

- Mettre en pratique les concepts appris en cours.
- Approfondir mes compétences en développement.
- Conserver une trace de ma progression.
- Expérimenter de nouvelles technologies et méthodologies.
